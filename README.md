# lopezpino.com

## Installation

```
sudo apt-get install ruby
sudo apt-get install ruby-dev
sudo apt-get install zlib1g-dev

sudo gem install bundler -v 1.13.7

sudo bundle install

```

Serve the website with jekyll:

```
sudo bundle exec jekyll serve
```
